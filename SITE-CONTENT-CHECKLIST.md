# 최종 사이트 내용 체크리스트

## 이미 확인된 정보

- 박원근 감독님과 함께 달리는 러닝크루
- 장소: 도안호수공원
- 정확한 집결 장소: 도안호수공원 모임 광장
- 집결 장소 안내 이미지 확보: `r15/assets/meeting-place-map-source.png`
- 신규 문의 카카오톡: 도안호수러닝크루/공식채널
- 카카오톡 오픈채팅 링크: https://open.kakao.com/o/gYG4iHwi
- 전화번호 공개 여부: 공개 사이트에는 전화번호를 표시하지 않고 카카오톡 공식채널만 안내
- 당근 모임: 도안호수러닝크루
- 당근 모임 링크: https://daangn.com/kr/share/community/ref/invite-group/bb1xY4o4p5E
- 인스타그램: 현재 미운영, 추후 개설 예정
- 네이버 밴드: 정회원 전용, 공개 사이트에는 링크를 표시하지 않음
- 정회원 회비: 월 10,000원
- 회비 사용 안내: 사용 내역을 매월 정회원 채널에 공지
- 정회원 혜택: 현재 확정 보류, 공개 사이트에는 임의로 표시하지 않음
- 참여 대상: 연령 제한 없음, 남녀노소 누구나 환영
- 박원근 감독님 소개: 전 국가대표 마라토너이자 대전시민 마라톤교실을 이끌어 온 지도자. 기본 자세부터 기록 향상, 마라톤 대회 준비까지 각자의 목표에 맞춰 꾸준히 달릴 수 있도록 함께 지도
- 박원근 감독님 공개 근거: 대전광역시 월간 대전이즈유 기사에서 국가대표 마라토너 활동, 한국 신기록 13차례 경신, 대전시민 마라톤교실 지도 내용을 확인
- 박원근 감독님 공개 사진 확보: `r15/assets/coach-park-wongeun-source.jpeg`
- 날씨 운영 기준: 호우·강풍·폭염·한파 특보 또는 노면 상태 등으로 안전한 훈련이 어렵다고 판단되는 경우 훈련을 변경하거나 취소할 수 있음
- 날씨 최종 판단·공지: 박원근 감독님이 최종 판단하고 카카오톡 공식채널에 안내
- 정기 러닝: 월·수·금·토
- 05:30~06:00 워밍업
- 06:00 그룹 러닝 시작
- 목표별 그룹 운영: 5km 도전, 하프코스 도전, 풀코스 도전 기준으로 안내
- 그룹 운영 안내 문구: 당일 컨디션과 목표에 맞춰 무리하지 않는 범위에서 함께 달림
- 07:30 전후까지 운영하되 개인 일정에 맞춰 자유롭게 마무리 가능
- 게스트 월 3회 이상 참석 → 신청서 작성 → 운영진 확인 → 정회원 진행
- 첫 참석자 준비물: 움직이기 편한 운동복, 러닝화 또는 가벼운 운동화, 개인 물 또는 물병
- 계절별 권장 준비물: 새벽 시간용 밝은색·반사 소재 용품, 여름 모자·자외선 차단제, 겨울 장갑·바람막이, 필요 시 작은 수건
- 몸 상태 안내: 몸 상태가 좋지 않거나 통증이 있다면 무리하지 말고 운영진에게 알림

## 공개 직전에 추가할 사이트 요소

- 카카오톡 링크 공유용 대표 이미지와 소개 문구
- favicon
- sitemap.xml
- 공개용 robots.txt
- 각 페이지의 제목·설명
- 자체 호스팅 한글 폰트
- 모바일·데스크톱 크기의 WebP/AVIF 이미지
- 실제 404 페이지 디자인
- 개인정보·사진 공개 범위 안내

## 최종 승인 기준

- 휴대폰에서 10초 안에 `누구 / 언제 / 어디서 / 초보 가능 여부 / 참여 방법`을 알 수 있음
- 모든 메뉴와 신청 버튼이 실제 주소로 연결됨
- 테스트·예시·placeholder 문구가 없음
- 운영진 내부 자료가 `public/`에 없음
- 작은 휴대폰에서도 가로 스크롤이 없음
- 카카오톡 내부 브라우저에서 정상 작동함
- `npm run check:release`가 통과함
