01 — FIRST SHOES

첫 러닝화, 이것만 알면 끝 👟

“무슨 신발 사요?”가 제일 많은 질문이에요. 처음엔 비싼 거 필요 없습니다. 딱 3가지만 기억하세요.

01

10만원대 쿠션화

카본화·경기화는 한참 나중에! 처음엔 푹신한 데일리 쿠션화면 충분해요.

02

꼭 신어보고

러닝화는 신는 순간 편해야 정답. ‘길들이는 기간’ 없습니다.

03

내 발에 맞는 것

최저가보다 내 발에 맞는 게 우선. 초반 부상을 확 줄여줘요.

👟
신발 종류 3가지
  • 쿠션화 — 푹신한 기본형. 초보 90% 정답.
  • 안정화 — 발이 안쪽으로 무너지는 분(과회내)용.
  • 카본화 — 경기용. 초보는 무릎·종아리 무리 ✗
📏
사이즈 ⚠️
  • 평소보다 살짝 여유 (엄지 한 마디 ≈ 1cm)
  • 신어볼 땐 저녁·걷고 난 뒤 (발 부었을 때)
  • 발볼 넓으면 와이드(2E) 확인
🔄
교체 시기
  • 대략 500~800km / 주3회 기준 6~9개월
  • 쿠션 꺼진 느낌 + 발·무릎 뻐근하면 신호
  • 마일리지 앱 쓰면 자동 계산 편해요

🙅 초보 흔한 실수 TOP 5

  1. 패션 스니커즈로 뛰기 (쿠션·구조가 달라요)
  2. 첫 신발부터 비싼 카본화 (부상 지름길)
  3. 사이즈 딱 맞게 사기 (발톱 멍·물집)
  4. 한 켤레로 매일 뛰기 (하루 쉬어야 쿠션 복원)
  5. 최저가만 보고 발에 안 맞는 거 사기

💰 예산별로 이렇게

  1. 입문 8~13만 — 브랜드 기본 데일리 쿠션 트레이너. 데카트론·푸마 가성비 라인 인기.
  2. 여유 15만~ — 상위 쿠션화. 처음부터 안 가도 OK.
  3. 일상+러닝 겸용 — 무난한 쿠션화 한 켤레로 충분.

※ 가격은 대략이에요. 세일·모델마다 다르니 구매 전 확인!

02 — SOCKS

신발 샀으면, 양말도! 🧦

“양말이 그게 그거 아냐?” 아닙니다. 신발만큼 물집을 좌우하는 숨은 핵심이에요. 면양말만 벗어나도 발이 확 편해집니다.

💧
왜 중요? — 면양말은 물집 주범
  • 면 100% = 땀 머금고 안 마름 → 축축 → 마찰 → 물집
  • 기능성 = 땀 배출 → 뽀송 → 물집 뚝
  • 신발 바꾸기 전에 양말부터 바꿔도 편해져요
🧵
뭘 신어요? — 소재·두께
  • 소재: 폴리에스터·나일론·메리노울 (면 ✗)
  • 두께: 여름·속도=얇게 / 장거리=두껍게 / 초보=중간
  • 발가락 물집 잦으면 → 발가락양말(인진지 계열)

✅ 물집 안 나는 4가지

  1. 딱 맞는 사이즈 (헐렁하면 쓸림)
  2. 봉제선 없는 발가락 (seamless)
  3. 뒤꿈치 힐탭 (뒤꿈치 쓸림 방지)
  4. 발가락 물집 잦으면 발가락양말

💡 실수 & 꿀팁

  1. 피하기: 면양말, 너무 큰 양말
  2. 양말도 소모품 — 여러 켤레 굴려 신기
  3. 물집 잘 나면: 마찰 부위 바세린 or 얇은 이중양말
