# 사진 교체 슬롯

현재 R5는 사진 없이도 레이아웃을 확인할 수 있도록 CSS 기반 중성 placeholder를 사용합니다.

실제 사진 준비 후 권장 파일:

- `hero-lake.webp`: 도안호수 전경 또는 인물이 식별되지 않는 러닝 장면, 가로 16:9 이상
- `activity-lake.webp`: 코스와 주변 환경을 보여주는 가로 사진, 가로 4:3 이상
- `detail-gear.webp`: 운동화·반사밴드·준비물 등 상세 사진, 가로 4:3 이상

이미지를 추가할 때 `.media-slot` 내부에 `<img>`를 넣고 `object-fit: cover`를 적용합니다. 실제 크루원이 식별되는 사진은 공개 동의를 확인한 뒤 사용합니다.
