# R6 Night Editorial

도안호수러닝크루 R4의 콘텐츠를 재구성한 정적 테스트 사이트입니다.

- 원본 R4 파일은 수정하지 않습니다.
- 일정, 장소, 인원, 페이스, 링크는 예시 또는 확인 전 정보입니다.
- 사진 슬롯은 실제 도안호수 사진을 나중에 교체하기 위한 자리입니다.
- `brief.html`은 공개 메뉴와 푸터에서 연결하지 않지만, 정적 파일이므로 URL 접근을 막는 보안 기능은 없습니다.

로컬 확인:

```bash
python3 -m http.server 4173 --directory .
```

그다음 `http://localhost:4173`을 엽니다.
